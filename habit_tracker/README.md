# Habit Tracker Web App

A clean, responsive, single-page habit tracker built entirely with Vanilla HTML, CSS, and JavaScript. 

No frameworks, no build steps, and no dependencies.

## Features
*   **Weekly Grid View:** Track habits week-by-week with a layout that starts on Monday.
*   **Smart Streak Calculation:** Calculates current consecutive daily streaks. It does not penalize you or break your streak if you haven't checked off today yet.
*   **Persistent Storage:** Data is saved automatically to your browser's `localStorage` so habits and checkmarks remain after refreshing the page.
*   **Responsive Design:** Features a mobile-friendly horizontal scrolling grid with a sticky column for habit names, ensuring usability on narrow phone screens up to wide desktop monitors.
*   **Historical Navigation:** Navigate between past and future weeks to view historical checkmarks.

## How to Run Locally

Because this project uses vanilla web technologies, there is nothing to install.

1. Download or clone this repository.
2. Open the folder containing the project files.
3. Double-click the `index.html` file to open it in your default web browser.
